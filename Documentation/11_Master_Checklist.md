## Phase 2: Company Registration (Weeks 9-12)

**Goal:** Legally register Pvt Ltd company in India

### Week 9: Pre-Registration
- [ ] Decide: **Private Limited Company** (for VC funding)
- [ ] Minimum 2 directors + 2 shareholders (can overlap)
- [ ] Think of 3-5 company names
- [ ] Check availability on MCA portal (mca.gov.in)
- [ ] Reserve .in and .com domains (‚Çπ500-1,000)
- [ ] Secure registered office (‚Çπ5-15K/month virtual office)

### Week 10: Get DSC & DIN
- [ ] Get DSC for all directors (‚Çπ1-2K each via eMudhra/Sify)
- [ ] Apply for DIN (Director ID Number) - Form DIR-3
- [ ] File RUN (Reserve Unique Name) on MCA
- [ ] Wait for name approval (1-2 days)
- [ ] Prepare: PAN, Aadhaar, address proof, photos for all

**Cost:** ‚Çπ2,000-4,000 (DSC for 2 directors)

### Week 11: SPICe+ Filing
- [ ] Draft Memorandum of Association (MOA)
- [ ] Draft Articles of Association (AOA)
- [ ] Collect KYC from directors/shareholders
- [ ] Get NOC from registered office owner
- [ ] File SPICe+ form on MCA portal
- [ ] Pay fees: ‚Çπ500-3,000 based on authorized capital
- [ ] Attach all required documents

**Cost:** ‚Çπ10,000-20,000 total (incl professional help)

### Week 12: Post-Registration
- [ ] Receive Certificate of Incorporation (COI)
- [ ] Receive PAN card (company)
- [ ] Receive TAN (Tax Deduction Number)
- [ ] Note CIN (Corporate Identification Number)
- [ ] Open current account (ICICI/HDFC/Axis recommended)
- [ ] Deposit initial capital (‚Çπ10K-1L)
- [ ] Apply for **DPIIT Startup India Recognition** (FREE!)
- [ ] Register for GST (if expecting >‚Çπ20L revenue)

**Milestone:** ‚úÖ Company legally incorporated in India

**Total Cost:** ‚Çπ10,000-20,000  
**Time:** 7-14 days actual processing

## Compliance Calendar (India) ‚ö°

### Monthly Deadlines

**7th of each month:**
- [ ] TDS deposit (previous month)

**11th of each month:**
- [ ] GSTR-1 filing (outward supplies)

**15th of each month:**
- [ ] EPF payment (if applicable)
- [ ] ESI payment (if applicable)
- [ ] Professional Tax (varies by state)

**20th of each month:**
- [ ] GSTR-3B filing + GST payment

### Quarterly
- [ ] **31st Jan/Apr/Jul/Oct:** TDS Returns (Form 24Q for salary, 26Q for others)
- [ ] **15th Jun/Sep/Dec/Mar:** Advance Tax payment
- [ ] **Board Meeting** (minimum quarterly)

### Annually
- [ ] **By Sept 30:** AGM (Annual General Meeting)
- [ ] **By end Oct:** File AOC-4 (within 30 days of AGM)
- [ ] **By end Nov:** File MGT-7 (within 60 days of AGM)
- [ ] **By Sept 30:** DIR-3 KYC for all directors
- [ ] **By Oct 31:** ITR filing (if audit required), July 31 otherwise
- [ ] **By Dec 31:** GSTR-9 (GST Annual Return)
- [ ] **Annual:** Statutory Audit
- [ ] **Renew:** Professional Tax, Shops & Establishment
